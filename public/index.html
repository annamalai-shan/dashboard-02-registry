dashboard-02

A SaaS dashboard starter — stat cards, a revenue chart, an activity feed, and a filterable data table. Install it into any shadcn/ui project: 

npx shadcn@latest add https://annamalai-shan.github.io/dashboard-02-registry/r/dashboard-02.json

▶ View the live demo

It includes:

  • Stat cards (KPI row)
  • Revenue area chart (recharts)
  • Recent activity feed
  • Sortable / filterable data table (TanStack Table)
  • App sidebar, nav, and site header shell
